# Rest Assured API Test Project

This project is developed as part of the Software Test Engineering course.

## Technologies Used
- Java
- Maven
- JUnit 4
- Rest Assured
- Node.js
- Express.js

## Project Description
The project includes:
- A simple REST API developed using Node.js and Express.js
- Automated API tests written using Rest Assured

## API Endpoints

### GET /users/1
Returns a sample user object.

### POST /users
Creates a new user using JSON request body.

## Test Scenarios
- Status code validation
- Response body validation
- Response time validation (less than 3 seconds)
