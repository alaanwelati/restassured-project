# REST Assured API Testing with Node.js and Express

This project includes:
- A simple REST API developed using Node.js and Express.js.
- Automated API tests written using REST Assured.

## API Endpoints

- GET /users/1: Returns a sample user object.
- POST /users: Creates a new user using JSON request body.

## Test Scenarios
- Status code validation
- Response body validation
- Response time validation (less than 3 seconds)

## How to Run the Project
1. Start the API
2. Run the tests using JUnit and RestAssured
